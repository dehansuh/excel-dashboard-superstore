# Excel Dashboard — Superstore Sales

Retail sales dashboard built with Excel using Superstore dataset (9,994 rows, 2014–2017).

## Key Findings
- West region most profitable (15% margin) — Central lowest (8%)
- Furniture margin only 2% despite high sales volume — root cause: 17.4% avg discount
- Revenue grew 51% (2014→2017) but profit margin flat — discount strategy needs optimization

## Tools
Excel — Power Query, Pivot Tables, Charts, KPI Cards

Kaggle — Superstore Sales Dataset
